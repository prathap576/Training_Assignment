web Development

HTML

HTML (HyperText Markup Language) is the standard language for creating and structuring web page content, using tags like for paragraph, for heading to define elements such as text, images, links, and lists, forming the essential skeleton of every website, often styled with CSS and made interactive with JavaScript. It's a markup language, not programming, that tells browsers how to display content meaningfully, creating the layout and hierarchy for everything you see online.

CSS

Cascading Style Sheets (CSS) is a style sheet language used to describe the presentation and styling of a document written in HTML or XML. It is a core technology of the World Wide Web, alongside HTML and JavaScript.

JavaScript

JavaScript is a programming language that enables interactive web pages. It is an interpreted, object-oriented language with first-class functions, and is one of the core technologies of the World Wide Web. JavaScript is used to create dynamic content and interactive features on websites.
